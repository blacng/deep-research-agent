# Technology Stack

## Frontend

*   **Framework:** Next.js (16.0.10), React (19.2.1)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **UI Components:** Radix UI, custom components
*   **Data Visualization:** Recharts, `@xyflow/react`
*   **Animation:** Framer Motion

## Backend

*   **Runtime:** Node.js (via Next.js API routes)
*   **Language:** TypeScript

## AI / API Integrations

*   **Large Language Models:** Anthropic Claude Agent SDK, Google Gemini (`@google/genai`)
*   **Search:** Exa-JS

## Other Libraries

*   **Form Management:** React Hook Form, Zod
*   **Markdown Rendering:** React Markdown
*   **Logging:** Winston
*   **Utilities:** class-variance-authority, clsx, lucide-react, tailwind-merge
